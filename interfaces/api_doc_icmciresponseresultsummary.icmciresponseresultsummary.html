Interface ICMCIResponseResultSummary

Represents the "resultsummary" field of the CMCI API response, parsed from XML to a javascript object with the xml2js package. See the following link for more information: https://www.ibm.com/support/knowledgecenter/SSGMCP_5.2.0/com.ibm.cics.ts.clientapi.doc/topics/clientapi_resultsummary_element.html
